Netflix_subset is the subset of the data provided by Netflix. 

R is the rating matrix. Only the data of the popular movies and movie addicted users are used in this dataset, so that R has no missing data. Thus you should devide R into training set and testing set.

uid is the user id corresponding to each column of R.
mid is the movie id corresponding to each row of R. Refer to movie_titles.txt to find the corresponding movie titles.
